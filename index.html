Summer Internship Project, Jadavpur University

Machine Learning based Keyword Extraction from Biomedical Documents
Under Dr. Kamal Sarkar

This project focuses on extracting the keywords (or phrases) pertaining to a set of medical documents. Biomedical articles from Open Access Media Journal were processed and used as data sets, which were then chunked into phrases with the help of GeniaTagger. The corpus used for this project was the Genia Corpus. Also, a domain specific feature set was developed to increase the accuracy. The influence of local variables as well as global variables were studied with the constructed Neural Network.
